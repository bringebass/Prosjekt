Visitar Wix

Iniciando

Regístrate

Registrarse en Wix es sencillo y quizás lo más importante, es gratis. Siempre que proporcione su dirección de correo electrónico y elija una contraseña, está listo para comenzar. Alternativamente, puede optar por iniciar sesión a través de Facebook o Google. Debe tener al menos trece años para inscribirse o dieciséis si vive dentro de la Unión Europea.

Plantillas

Cuando inicia el creador de sitios web de Wix, le preguntan que tipo de sitio desea crear y usted puede elegir entre las siguientes doce categorías: negocios, diseñador, blog, tienda en línea, restaurantes y comida, belleza y bienestar, fotografía, fitness, portafolio y CV, música, eventos u “otros”. Basado en su elección, Wix le presenta una serie de plantillas adecuadas para la categoría que escogió específicamente. Cuando decidimos construir un sitio fitness con wix, nos presentaron 49 diferentes plantillas para esto. La mayoría de estas plantillas tenían un buen diseño y contenían una gran parte del contenido y funcionalidades que se busca al crear un sitio web. Entre otras cosas, encontrará botones de redes sociales, barras de navegación, burbujas de chat e imágenes de fondo. Las plantillas brindan un excelente punto de partida para la creación de su sitio web.

Wix ADI

Wix ha agregado una característica realmente interesante a su creador de sitios web llamado Wix ADI. Wix ADI es un servicio donde se utiliza inteligencia artificial para ayudarle a crear un sitio web. Al responder una serie de preguntas sobre el sitio que desea, WIX ADI continúa de manera autónoma y crea un sitio web para usted. Supongamos que quiere un sitio para su restaurante, entonces Wix ADI le haría preguntas como “¿Qué necesita en su nuevo sitio web?”, o “elija un tema que le guste”. Una vez completadas todas las preguntas (o de hecho, las que desea responder) Wix ADI abre un nuevo sitio web para usted. ¡Ciertamente nunca hemos visto una manera más fácil de crear un sitio web! Es verdad que probablemente necesite realizar algunos cambios al sitio que fue creado para usted, pero ADI hace gran parte del trabajo.

Construyendo su sitio web

Interfaz de usuario

Ya sea que haya decidido utilizar Wix ADI o simplemente escoger una plantilla propia, la interfaz de usuario para realizar cambios a su sitio web es la misma. Es una interfaz gráfica de fácil comprensión para cualquiera con nociones básicas de informática. Un montón de elementos (menú, imagen de fondo, formulario de contacto etc.) es visible en una página en blanco y usted es libre de moverlos, intercambiarlos, cambiar el color, agregar funcionalidades y diseñar la página más o menos exactamente como la desea. Una cuadrícula visible divide la página en nueve partes (izquierda, centro, derecha del encabezado, principal y pié de página) así siempre sabrá exactamente en qué parte coloca un elemento. Esto es importante, no sólo para el diseño, sino también cuando se trata de propiedades receptivas. Si usted es nuevo en la creación de una web, este puede ser un término desconocido, por lo que le daremos una explicación rápida. Una página web receptiva es aquella que se adapta a diferentes dispositivos con diferentes anchos de pantallas. En el mundo actual, donde muchas personas están usando sus teléfonos móviles para navegar por internet, es absolutamente vital asegurarse que su página web responda (relájese, WIX se encarga de esto).

Agregar elementos

Agregar nuevos elementos a su sitio web se hace a través de un menú en el lado izquierdo de la página. El menú se divide en las siguientes categorías: Menú y páginas, fondo, agregar, administrador de tema, agregar aplicaciones, medios, mi blog, reservas, administrador de contenido y herramientas de negocios Ascend. Algunos de los elementos del menú se utilizan para añadir elementos a su sitio web, mientras otros son usados por ejemplo para crear una publicación de blog o agregar una aplicación a su sitio. Cuando se trata de agregar elementos en un lateral (por elementos nos referimos por ejemplo a un cuadro de navegación o un nuevo encabezado) puede escoger en un rango de estos. Lo mejor de muchos de los elementos que puede añadir es que su funcionalidad ya está implementada. Si usted por ejemplo desea agregar un formulario de envío, no tiene que hacer nada más que colocarlo en su sitio web en alguna parte y la funcionalidad del formulario de envío ya está en su lugar. También es posible estilizar los elementos tanto en términos de diseño como de funcionalidad. Por ejemplo, puede querer cambiar el tamaño de un botón o asegurarse de que el botón cambie de color cuando alguien pase el mouse sobre él.

Aplicaciones

Mencionamos que puede agregar aplicaciones a su sitio web con el creador de sitios web WIX. Esto se hace a través del mercado de aplicaciones Wix. Un ejemplo de una aplicación que puede añadir es la aplicación Wix Stores. Por medio de ésta puede crear una tienda virtual profesional para su sitio web. Cosas como transacciones seguras, gestión de pedidos e integración de redes sociales ya están incluidas en la aplicación. Esto literalmente hace posible que cualquiera pueda crear su propia tienda en línea. Otro ejemplo de una aplicación que puede agregar es la aplicación Bablic, que facilita la creación de un sitio web multilingüe. Hay muchas aplicaciones que puede agregar a su sitio web a través de la aplicación Wix Stores. Lo bueno es que puede visualizar como las diferentes aplicaciones son revisadas por los clientes, de esta manera previene terminar con una aplicación inútil o que no funciona bien. La capacidad de añadir aplicaciones de calidad a su sitio web es algo que encontramos realmente valioso con el creador de sitios web Wix.

Herramientas SEO

Si está planeando construir un sitio web en 2020 es importante que considere el SEO. SEO significa optimización de motores de búsqueda y es lo que hace que su sitio web aparezca en diferentes motores de búsquedas (por ejemplo en google.com). Wix tiene muchas herramientas diferentes para impulsar su SEO, le ayuda a crear palabras clave, a clasificar su sitio web y se asegura de que tenga marcado semántico en su código. Si lo desea, también puede contratar a un experto en SEO a través de Wix. El hecho de que Wix haya incluido herramientas de SEO tan poderosas en su creador de sitios podría ser una de las razones por las que tantas personas usan este creador de sitios web.

Conclusión

El creador de sitios web Wix es fácil de usar, tienen plantillas de aspecto profesional y una gran funcionalidad. Apreciamos especialmente la capacidad de agregar aplicaciones a su sitio web, al igual que hacer uso de sus herramientas profesionales de SEO. Es difícil encontrar cosas negativas sobre este creador de sitios web, pero si estuviésemos forzados a buscar una, sería el hecho de que Wix no ofrece un chat de atención al cliente. Sin embargo, esto no es un problema digno de mencionar ya que puede fácilmente contactarlos por teléfono. Adicionalmente, encontrará las respuestas a la mayoría de las preguntas que pueda tener en su página web de atención al cliente.
