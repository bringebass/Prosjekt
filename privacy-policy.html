Privacy Policy

Thewebsitecentral.com

PLEASE READ THE INFORMATION BELOW CAREFULLY. IT CONTAINS IMPORTANT INFORMATION REGARDING OUR PRIVACY POLICY

Our privacy policy was last updated on September 27, 2020.

1. Google analytics

1.1 We collect, store and process data on this website through the use of google analytics.

1.2 Google analytics is a tool delivered by the company Google. Briwide AS (referred to as “the company”) have an agreement with Google that enables us use of this service.

1.3 Google analytics uses cookies. This means that our webiste uses cookies through google analytics. By using our website you consent to this usage of cookies.

2. How google analytics is implemented on our website

2.1 Google analytics is implemented through a JavaScript tracking code (referred to as “the tracking code”) in the HTML for www.thewebsitecentral.com (referred to as “the website”).

2.2 The tracking code is found on all HTML-files related to the website.

3. How google analytics work

3.1 On every occasion that a user visits the website, the JavaScript tracking code mentioned under point 2.1 collects anonymous data about:

  • How the user behaved on the website (e.g. which links or buttons where clicked, or how long the user spent on the website, how the user navigated on the website).
  • Browser, device, language, and operating system of the user.
  • Number of visits to the website.
  • Traffic source (e.g. did the user get into the website through a search engine, or an article on Facebook).
  • Other relevant data.

3.2 The tracking code sends updated data continuously while the user is active on the website. After 30 minutes of user inactivity, the tracking code starts a new session of user data collection. If the user leaves the website, a new session is started on revisit.

3.3 After collecting data, the tracking code sends this information to google analytics where the data is processed.

3.4 Google analytics displays detailed reports of the processed data which is visible to the company.

3.5 Google stores the collected data for a maximum of 26 months.

4. Why we gather the data

4.1 There is different reason of why we gather the data mentioned under point 3.1:

  • It enables us to enhance user experience.
  • It enables us to evaluate our website` popularity within different user groups.
  • It enables us to enhance visitor conversion (numbers of visitors who end up clicking at our links and buttons).
  • It gives is information about how we can improve our results within the marketing and affiliate industry.

5. Who is this data shared with?

5.1 The data mentioned under point 3.1 is shared only within the company.

5.2 Because Google delivers this service, they will have access to the data.

5.3 We do not share the data to anybody outside of the company.

6. Affiliate

6.1 The website provides affiliate links. This means that the affiliate companies we collaborate with, will be able to pinpoint the website as the traffic source for customers visiting their websites.

6.2 The affilate companies we collaborate with uses cookies. This means that our website uses cookies through the affiliate companies. By using our website you consent to this usage of cookies.

6.3 The company will be able to view the number of people who signs up as customers at our collaborating affiliate companies.

7. Email messages sent to the company

7.1 Emails sent to the company will be deleted within 45 days. If you want it deleted at an earlier stage, contact us (see point 8).
