What is webhosting?

Web hosting can be thought of as a hotel. It is essentially a service, hosted on a web server, which makes it possible for a user to publish files on the internet. One example of this is publishing homepages. The location of the web host is usually tied to a domain name and usually includes emails. This web host is also operated by a control panel with a graphical user interface made available over the internet. The most common type of hosting offers several users to share their utilities, known as shared hosting, by letting each customer share the resources of a server. Types of webhosting

  • Shared hosting: The budget webhotel. Share your server with other users at a lower rate.
  • Virtual private server (VPS) hosting: The higher end hotel, with your own utilities at a higher rate.
  • Dedicated server hosting: The most expensive option, which gives you complete ownership of the host.
  • Cloud hosting: This type of hosting makes resources available through several cloud servers and is generally hosting the application or website through a network of cloud servers.
  • Managed hosting: here you outsource the management of your application, and instead of just paying for the server hosting, you pay additionally for services such as 24/7 support, updates and maintenance of your application or website.
  • Colocation: A physical data centre that allows for you to use your own hardware to host a server, while they provide the space and maintenance for a fee.
