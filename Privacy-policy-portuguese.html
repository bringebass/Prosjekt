Política de Privacidade

Thewebsitecentral.com

POR FAVOR, LEIA AS INFORMAÇÕES ABAIXO ATENTAMENTE. ELAS CONTÉM INFORMAÇÕES IMPORTANTES SOBRE NOSSA POLÍTICA DE PRIVACIDADE

Nossa política de privacidade foi atualizada pela última vez em 27 de setembro de 2020.

1. Google analytics

1.1 Coletamos e processamos dados neste site por meio do Google Analytics.

1.2 O Google Analytics é uma ferramenta fornecida pela empresa Google. A Briwide AS (referida como “a empresa”) tem um contrato com o Google que nos permite usar este serviço.

2. Como o Google Analytics é implementado em nosso site

2.1 O Google Analytics é implementado por meio de um código de rastreamento JavaScript (referido como “o código de rastreamento”) no HTML para www.thewebsitecentral.com (referido como “o site”).

2.2 O código de rastreamento é encontrado em todos os arquivos HTML relacionados ao site.

3. Como funciona o Google Analytics

3.1 Em todas as ocasiões em que um usuário visita o site, o código de rastreamento JavaScript mencionado no ponto 2.1 coleta dados anônimos sobre:

  • Como o usuário se comportou no site (por exemplo, quais links ou botões foram clicados, ou quanto tempo o usuário passou no site, como o usuário navegou no site).
  • Navegador, dispositivo, idioma e sistema operacional do usuário.
  • Número de visitas ao site.
  • Origem do tráfego (por exemplo, o usuário acessou o site por meio de um mecanismo de pesquisa ou de um artigo no Facebook).
  • Outros dados relevantes:

3.2 O código de rastreamento envia dados atualizados continuamente enquanto o usuário está ativo no site. Após 30 minutos de inatividade do usuário, o código de rastreamento inicia uma nova sessão de coleta de dados do usuário. Se o usuário sair do site, uma nova sessão será iniciada na revisita.

3.3 Depois de coletar os dados, o código de rastreamento envia essas informações ao google analytics, onde os dados são processados.

3.4 O Google analytics exibe relatórios detalhados dos dados processados que são visíveis para a empresa.

4. Por que coletamos os dados

4.1 Existem diferentes razões pelas quais coletamos os dados mencionados no ponto 3.1:

  • Isso nos permite aprimorar a experiência do usuário.
  • Isso nos permite avaliar a popularidade do nosso site dentro de diferentes grupos de usuários.
  • Permite-nos aumentar a conversão de visitantes (número de visitantes que acabam por clicar nos nossos links e botões).
  • Ele fornece informações sobre como podemos melhorar nossos resultados no setor de marketing e afiliados.

5. Com quem esses dados são compartilhados?

5.1 Os dados mencionados no ponto 3.1 são compartilhados apenas dentro da empresa.

5.2 Como o Google oferece esse serviço, eles terão acesso aos dados.

5.3 Não compartilhamos os dados com ninguém fora da empresa.

6. Afiliado

6.1 O site fornece links de afiliados. Isso significa que as empresas afiliadas com as quais colaboramos serão capazes de apontar o site como a fonte de tráfego para os clientes que visitam seus sites.

6.2 O site fornece links de afiliados. Isso significa que as empresas afiliadas com as quais colaboramos serão capazes de apontar o site como a fonte de tráfego para os clientes que visitam seus sites.

7. Mensagens de e-mail enviadas para a empresa

7.1 Os e-mails enviados para a empresa serão excluídos em 45 dias. Se você deseja excluí-lo em um estágio anterior, entre em contato conosco (ver ponto 8).
